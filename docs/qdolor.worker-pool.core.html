WorkerPool

protocol

A pool of concurrent workers.

members

start!

(start! this ctx)

Start the worker pool. ctx is an arbitrary map injected into every task callback during this pool’s lifetime. Returns nil.

stop!

(stop! this)

Signal all workers to stop and block until they have exited their current iteration. Returns nil.
